***Лабораторная работа №2***  
  
Создание рендера с использованием VBO, VAO и EBO
  
Задание №1  
Вместо того, чтобы передавать один цвет во фрагментный шейдер, создайте зависимость изменения цвета от времени.  
Далее Вам необходимо рассчитать от некоторого закона изменение цвета.
Удобнее всего использовать периодические законы. Как пример: sin или cos – в этих функциях
независимо от входящего значения возвращаемое значение будет находится в диапазоне от -1 до 1.
  
Задание №2  
На основе написанных элементов соберите свою библиотеку для загрузки шейдерных программ из файлов проекта.
Добавьте функционал для пересылки uniform переменных в одну строку (включите в созданную Вами библиотеку).  
В идеальном варианте посмотрите какие есть варианты по количеству элементов и типу данных.
